# Term Sheet Review Checklist

## Economics

### Valuation
- [ ] Pre-money valuation stated
- [ ] Compare to market comps
- [ ] Implied ownership percentage calculated
- [ ] Fully-diluted cap table provided

### Liquidation Preference
- [ ] Multiple stated (1x standard)
- [ ] Participating vs non-participating identified
- [ ] Participation cap (if participating)
- [ ] Seniority/parity with prior rounds

### Anti-Dilution
- [ ] Type specified (weighted average standard)
- [ ] Broad-based vs narrow-based
- [ ] Carve-outs for option pool refresh

### Option Pool
- [ ] Size specified (10-15% typical)
- [ ] Pre or post-money (post = founder dilution)
- [ ] Increase included in round?

### Dividends
- [ ] Cumulative or non-cumulative
- [ ] Rate if applicable
- [ ] When payable

## Control

### Board Composition
- [ ] Number of seats specified
- [ ] Investor seats reasonable
- [ ] Observer rights (if any)
- [ ] Independent director requirements

### Protective Provisions
- [ ] List is reasonable length (5-10 items)
- [ ] No unusual blocking rights
- [ ] Threshold for consent reasonable

### Drag-Along
- [ ] Threshold specified (majority typical)
- [ ] Floor price protections
- [ ] Terms of sale requirements

### Tag-Along
- [ ] Pro-rata participation rights
- [ ] Same terms as selling shareholders

## Founder Terms

### Vesting
- [ ] Schedule specified (4yr/1yr cliff standard)
- [ ] Acceleration on change of control
- [ ] Reset or continuation from existing

### Repurchase Rights
- [ ] Repurchase at FMV or cost
- [ ] Lapse schedule

### Non-Compete/Non-Solicit
- [ ] Duration reasonable (12-24 months)
- [ ] Geographic scope reasonable
- [ ] Triggered only on termination

## Information Rights

### Financial Reporting
- [ ] Monthly/quarterly financials
- [ ] Annual audited statements
- [ ] Budget and forecasts

### Board Meeting Rights
- [ ] Meeting frequency specified
- [ ] Information package requirements

### Inspection Rights
- [ ] Access to records
- [ ] Reasonable limitations

## Other

### Pro-Rata Rights
- [ ] Who has rights
- [ ] Super pro-rata included?
- [ ] Pay-to-play provisions

### Registration Rights
- [ ] S-3 rights
- [ ] Piggyback rights
- [ ] Lock-up periods

### Expenses
- [ ] Lead investor legal fees capped
- [ ] Company expenses reasonable

### Exclusivity/No-Shop
- [ ] Duration reasonable (30-60 days)
- [ ] Fiduciary out clause

### Closing Conditions
- [ ] Standard legal conditions
- [ ] Employment conditions
- [ ] Material adverse change clause

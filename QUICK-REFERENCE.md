# Crowley Capital Diligence Tool — Quick Reference

## Commands Cheat Sheet

```
/diligence init <company>     # Start new deal
/diligence analyze --all      # Run all analysis
/diligence risks              # Generate scorecard
/diligence export             # Package data room
/diligence compare <a> <b>    # Compare deals
```

## Key Metrics Quick Reference

### Unit Economics
- **LTV:CAC > 3x** — Healthy unit economics
- **CAC Payback < 18mo** — Efficient acquisition
- **Gross Margin > 70%** — True SaaS margins

### Retention
- **NRR > 110%** — Growing without new customers
- **Gross Churn < 2%** — Strong product-market fit

### Efficiency
- **Burn Multiple < 2x** — Efficient growth
- **Runway > 18mo** — Adequate capital
